# Как запустить?
## 1. Установить Python
## 2. Создать venv
## 3. Скачать все библиотеки
```commandline
pip install -r requirements.txt
```
## 4. В переменных среды задать переменную (можно через файл .env) API_KEY
## 5. Запустить файл app.py
```commandline
python main.py
```
# Использование
## Сервис доступен по адресу http://127.0.0.1:8000
